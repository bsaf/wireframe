# Wireframe

Static site framework.

## Prerequisites

- Sass: `gem install sass`
- CoffeeScript

## Usage

1. Clone the repo
2. Name the project
3. Delete the .git folder and create a new one for this project

## Server and watchers

coffee --compile --watch js/app.coffee
sass --watch css/scss:css
python -m SimpleHTTPServer
